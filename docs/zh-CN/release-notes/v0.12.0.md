# v0.12.0 

_发布日期： 2026-02-05_

### 新增

- 全新的品牌 Logo。
<img width="402" height="396" alt="new-logo" src="https://assets.lbctrl.com/uploads/80ab481b-d840-488a-beb8-d424bdc85d75/new-logo.png" /><img width="402" height="396" alt="new-logo-dark" src="https://assets.lbctrl.com/uploads/423cb07b-8b1e-424f-b3d8-3210147ce108/new-logo-dark.png" />

- **指标**: 支持自定义技术指标配置和编辑。
<img width="1704" height="1144" alt="panel" src="https://assets.lbctrl.com/uploads/0138b4a1-5a59-4eea-9e15-8d03b087c557/indicators.png" />

- **图表**: 优化指标加载器，支持「参数」和「颜色」设置。
<img width="892" height="712" alt="setting" src="https://assets.lbctrl.com/uploads/37bc3d32-b43f-4bfd-84a2-cabf31be1e84/setting.png" />

- **资产**: 新增「现金明细」面板。
- 新增「订单」、「持仓」导出功能。

### 优化

#### 交易

- 增强交易键盘操作体验，支持使用键盘快捷键进行交易操作。
- 优化成交通知体验，仅推送到当前激活窗口，避免重复通知。
- 补充「当日订单」自动刷新功能，避免推送异常导致的订单更新可能不及时的问题。
- 「历史订单」添加默认状态过滤，默认展示「已成交」状态。
- 修正交易预估金额信息显示错误。

#### 行情

- 修复期权图表价格变动时可能不更新的问题。
- 投资组合面板移除当前账户订单筛选选项。

#### 资产

- 调整资产侧边栏最小宽度。

#### 其他

- 优化网络异常处理，避免意外的登出。
- 修复表格设置重置功能失效的问题。
- 统一分析模块中的中性色显示。
- 热力图优化，调整部分样式细节和悬停边框样式。
- 自动更新优化，使用进度圆圈显示下载进度。
- 修复部分系统配置拉取异常的问题。
- 修复特殊交易状态的涨跌颜色显示错误的问题。
