# v0.5.0-preview.1  <Badge type="warning" text="preview" />

_发布日期： 2025-07-24_

### 优化

- 修复全局历史部分期权数据未显示股票名称的问题，提升数据清晰度和使用体验。
- 修复登录模块的警告问题，提升认证过程的可靠性。
- 优化热力图保留基于涨幅范围的树状图展示，提升数据可视化清晰度。
- 改进鼠标点击的热点区域检测，增强交互流畅度和精准度。
