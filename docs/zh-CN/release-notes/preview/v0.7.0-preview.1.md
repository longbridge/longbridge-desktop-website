# v0.7.0-preview.1  <Badge type="warning" text="preview" />

_发布日期： 2025-09-04_

### 优化

- 优化验证交易密码弹窗样式，增强可读性与使用体验。
- 优化语言偏好设置修改后可实时刷新界面语言。
- 修复登录窗口未检查更新状态的问题。
- 修复资产持仓盈亏字段排序不准确的问题。
- 修复自动更新功能在部分场景下界面显示异常的问题。
- 修复 Windows 系统部分场景引发的应用异常退出问题。
