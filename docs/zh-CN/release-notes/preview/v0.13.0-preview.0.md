# v0.13.0-preview.0  <Badge type="warning" text="preview" />

_发布日期： 2026-02-26_

### 新增

- 新增全局设置面板。
- **个股**：新增估值分析面板 。
- **自选**：新增自选股多图表功能。
- **行情**：期权支持延长时段（盘前）。
- **交易**：当日订单支持「快捷改单」。

### 优化

- 调整用户菜单至标题栏，方便查看当前用户信息。
- 优化交易键盘下单操作。
- 更新交易表单样式细节。
- 优化行情特殊交易状态显示。
- 自选股过滤空分组。
- 优化热力图样式细节。
