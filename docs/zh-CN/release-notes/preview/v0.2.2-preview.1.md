# v0.2.2-preview.1  <Badge type="warning" text="preview" />

_发布日期： 2025-05-16_

## 新增

- 新增在 macOS 上使用 Cmd-1 至 Cmd-9 快捷方式切换主标签页，使用 Ctrl-1 至 Ctrl-9 快捷方式切换股票详情页。
- 新增针对空头交易图表的工具提示。
- 新增订阅期权深度推送数据功能。

## 优化

- 优化图表数据加载逻辑，当交易状态发生变化时能正确重新加载图表。
- 优化图表绘制，支持 GPUI 渲染路径以改善性能。
  - 优化指数图表绘制。
  - 优化流量图表绘制。
  - 优化分布图表绘制。
  - 优化空头交易和持仓图表。
- 优化初始选中股票列表逻辑。
- 优化面板尺寸调整问题，改善用户体验。
