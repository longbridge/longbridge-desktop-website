# v0.10.0-preview.1  <Badge type="warning" text="preview" />

_发布日期： 2025-11-27_

### 优化

#### 图表

- 优化了图表设置界面的交互样式，提供更直观的图表配置体验。
- 修复了图表鼠标右键点击时没有响应的问题。

#### 交易

- 优化了今日订单面板中的改单弹窗以独立的窗口显示，并支持自由的拖拽操作。
- 优化了当日成交统计面板交易金额显示逻辑，让不同市场不同币种的金额显示更加合理。
- 修复了改单时余额信息预览不正确的问题。
- 修复了买卖盘有行情变动时内容闪烁的问题。

#### 主题

- 优化了默认主题及其他主题的配色，让界面看起来更加美观。
- 优化了登录界面的背景图片，提升用户登录时的视觉体验。
