# v0.8.0-preview.3  <Badge type="warning" text="preview" />

_发布日期： 2025-10-17_

### 优化

- 优化了盘口面板的加载体验，解决数据加载时可能出现的布局抖动问题，使界面更加稳定流畅。
- 优化了财务报表列表的展示样式，使信息呈现更加清晰易读。
- 优化了全局历史记录列表中股票名称的显示，解决长名称可能出现的溢出问题。
- 修复了登录界面使用反馈按钮时导致应用异常退出的问题。
- 修复了部分场景下界面无法滑动的问题。
