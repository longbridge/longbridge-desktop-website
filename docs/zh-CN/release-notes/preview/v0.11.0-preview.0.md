# v0.11.0-preview.0  <Badge type="warning" text="preview" />

_发布日期： 2025-12-11_

### 新增

#### 多窗口支持关键资产功能

在「资产」标签页中出入金、货币兑换、资金记录等在内的核心功能，现支持使用独立窗口形式打开，以提升操作效率且不干扰主窗口的使用体验。

#### 全新的资产页验证交互

当你切换到「资产」标签页验证交易密码或者你未入金、未开户时会显示一个更精简的界面，以优化现在同一个标签页多个面板同时显示重复校验信息的不佳体验。

#### 一键查看当前股票订单与持仓

应用现在支持在当日订单、历史订单、持仓等面板右上角，点击当前标签浏览的个股按钮。一键筛选出个股的对应的订单和持仓信息，以优化现在需要多次点击才能查到的个股信息。

#### 其他

- 优化订单筛选时提供快捷筛选期权的功能，使用户可以更快速地在股票和期权之间切换。
- 优化了小尺寸屏幕下的行情面板的显示效果，保持最小的显示空间，避免内容被压缩。
- 「资产」-「当日成交統計」新增了对加密货币账户的支持。
- 修复了在 Linux 系统应用在部分场景异常退出的问题。
- 修复了市价订单的订单详情历史价格显示问题。
