# v0.13.0-preview.1  <Badge type="warning" text="preview" />

_发布日期： 2026-03-05_

### 新增

- **资产**：全新的「资金明细」。

### 优化

- 修复 Windows 平台可能出现的启动异常问题。
- 修复 Windows 平台标题栏按钮点击失效的问题。
- 「历史订单」新增「方向」筛选项。
- 优化「历史订单」日期范围默认值。
- 优化股票选择下拉框键盘操作。
- 修复「期权链」盘前价格显示。
- 「持仓」表格默认按市值排序。
- 修复部分二次确定框键盘快捷键失效的问题。
