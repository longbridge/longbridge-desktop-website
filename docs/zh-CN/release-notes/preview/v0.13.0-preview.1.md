# v0.13.0-preview.1  <Badge type="warning" text="preview" />

_发布日期： 2026-03-05_

### 新增
- 增加在 Title Bar 中的遮挡方法。
- 多图表支持使用 `WeakEntity`。
- 更新 `QuoteAppConfig` 的刷新机制。

### 优化
- 优化 Action Bar 中长昵称的显示。
- 修改 Counter Search 的下拉宽度及锚点方向。
- 更新订单显示范围为昨天。
- 为过去的订单表格添加操作筛选器。

### 其他
- 初始化 WebSocket 在异步应用上下文更新后运行。
- 修复贸易问卷确认对话框无法显示的问题。
- 更新用户反馈描述内容。
