# v0.9.0-preview.1  <Badge type="warning" text="preview" />

_发布日期： 2025-11-06_

### 优化

* 优化期权链表格的双击跳转功能，便于快速切换至图表视图进行技术分析。
* 优化自选股列表在不同窗口大小下的显示效果，提升浏览体验。
* 优化自定义标签页的滚动条显示逻辑，使界面更简洁。
* 优化自定义标签页中面板的圆角显示，提升界面一致性。
* 优化表格滚动响应速度，滚动操作更即时生效。
* 调整附加订单时间选择器的显示位置，使其居中呈现。
* 修复部分场景下弹窗关闭后界面无法响应的问题，确保操作流畅不中断。
* 修复登录界面在登录信息过期后未能提示重启的问题，提升登录体验可靠性。
