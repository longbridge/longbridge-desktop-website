# v0.1.31-preview.9  <Badge type="warning" text="preview" />

## 新增

- 优化股票行业列表排序，显示更清晰。
- 股票市场状态栏支持地区筛选。

## 优化

- 改进交易详情页面，使信息展示更加精确与清晰。
- 更新市场指数弹出菜单的定位方式，更加精准。
- 修复交易取消后无法恢复有效状态的问题。
- 提升市场走势和热图的提示信息展示效果。
- 将技术指标的开关功能移到侧边栏，优化操作流程。
- 从服务器端获取股票名称确保名称正确无误。
- 优化图表上的做空 Y 轴标签，将精度调整为小数点后 3 位。
