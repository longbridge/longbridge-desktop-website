# v0.11.0-preview.1  <Badge type="warning" text="preview" />

_发布日期： 2025-12-18_

### 优化

- 持仓面板新增买入 / 卖出快捷交易按钮，现在你可以在持仓面板直接发起交易。
- 优化了表格列宽设置的持久化功能，你对表格的设置会被保存并在下次打开时自动恢复。
- 美股期权新增了在盘前交易时间的交易支持。
- 优化了行情等级变化时的数据准确性。
- 优化了个股图表中价格更新速度，现在更快更准确。
- 修复了出入金、货币兑换等功能需要验证交易密码时未及时提示用户的问题。
- 修复了 Linux 或 Windows 上登出失败的问题。
