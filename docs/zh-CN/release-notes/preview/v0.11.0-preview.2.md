# v0.11.0-preview.2  <Badge type="warning" text="preview" />

_发布日期： 2025-12-31_

### 新增

- 个股「财务」支持查看更多历史数据。
- 新增「关注列表」分组管理。

### 优化

- 网络恢复后自动刷新期权链。
- 交易表单渲染优化，避免抖动。
- 修复 WebView 登出后没法正确重启客户端的问题。
- 调整深度视图样式细节。
- 图表价格推送时更新数据结束时间。
- 浏览器模块增加刷新功能，在移除 WebView 时保证功能一致。
- 盈亏趋势图支持图表动态高度调整，提高渲染效率。
- 修复 WebView 关闭后 Action Bar 丢失焦点的问题。
- 更新「快捷交易」快捷键，`cmd + t`。
