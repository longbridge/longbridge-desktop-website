# v0.1.31-preview.14 <Badge type="warning" text="preview" />

## 新增

- 支持交易切换股票时重置订单信息。
- 支持多种类型窝轮牛熊证订单。
- 当用户状态为 `暂停`、`休眠` 时提供明确的非可交易原因描述。

## 优化

- 优化模拟账户交易时不再需要验证交易密码。
- 交易数量估算金额支持千分位分隔显示。
- 修复交易数量组件点击事件不响应的问题。
- 禁用交易密码验证弹窗的点击弹框外部区域时关闭行为。
- 修复持仓 Tabs 顺序偶现混乱的问题。
- 优化持仓成本价格为 0 时显示内容应该用 `--`。
- 隐藏期权的交易类型菜单。
- 优化 `eps` 字段显示精度。
- 优化图表的单位提示信息。
- 支持图表根据股票类型显示不同的范围。
- 模拟账户不再显示入金引导信息。

## 其他

- 修复 `submarket`字段在报价配置中的显示问题。
- 更新订单详情描述标签宽度。
