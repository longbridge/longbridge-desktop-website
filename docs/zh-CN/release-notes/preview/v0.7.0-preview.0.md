# v0.7.0-preview.0  <Badge type="warning" text="preview" />

_发布日期： 2025-08-28_

### 优化

- 用户资产持仓模块支持展示期权类标的信息。
- 修复自动更新弹框中部分场景下文字闪烁的问题。
- 图表设置支持在独立窗口打开，增强个性化配置效率。
- 市场异动设置界面采用全新样式，提升视觉一致性。
- 修复交易模块价格字段未自动填充最新成交价的问题。
