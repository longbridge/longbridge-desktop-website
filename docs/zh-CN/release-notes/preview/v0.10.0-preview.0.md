# v0.10.0-preview.0  <Badge type="warning" text="preview" />

_发布日期： 2025-11-20_

### 新增

- **资产**: 在资产标签页新增「当日成交统计」面板，帮助您更直观地掌握今日盈亏数据。

### 优化

#### 交易

- 优化了交易窗口显示方式，从弹窗模式改为独立窗口模式，提供更灵活的多屏幕操作体验。
- 优化了下单后的状态通知机制，确保在交易激活后才进行提醒。
- 修复了订单重复错误通知的问题，避免误导性的重复提醒。
- 修复了订单筛选器中清除股票代码不生效的问题。

#### 账号与登录

- 优化登录界面国家/地区选择列表搜索功能，使查找和选择国家或地区更加便捷。
- 修复了登录窗口关闭时更新状态异常的问题。

#### 其他

- 优化了分时图背景透明度细节，提升视觉效果。
- 优化了自由布局标签页初始化时机，提升自定义工作区的加载速度。
- 应用更新版本时显示了成功和失败的状态图标，让更新进度更直观。
- 修复了网络重连后市场指数行情详情未重新加载的问题，确保数据始终保持最新。
