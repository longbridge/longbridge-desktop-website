# v0.12.0-preview.0  <Badge type="warning" text="preview" />

_发布日期： 2026-01-22_

### 新增

- **资产**: 新增「现金明细」面板。
- **图表**: 指标加载器优化，支持「参数」和「颜色」设置。
- 新增「订单」、「持仓」导出功能。

### 优化

- 修复期权图表价格变动时可能不更新的问题。
- 优化网络异常处理，避免意外的登出。
- 改进自选列表分组管理器界面。
- 修正交易预估金额信息显示错误。
- 优化自动更新交互，显示安装进度。
- 优化图表手势交互，避免意外的抖动。
- 优化界面文案，统一术语。
- 修复表格设置重置功能失效的问题。
- 在持仓面板中禁用当前账户订单筛选。
- 修复部分系统配置拉取异常的问题。
- 修复特殊交易状态的涨跌颜色显示错误的问题。
