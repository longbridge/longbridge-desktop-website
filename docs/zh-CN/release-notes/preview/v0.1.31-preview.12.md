# v0.1.31-preview.12  <Badge type="warning" text="preview" />

## 新增

### 交易
- 支持 `IOC` 和 `FOK` 有效期限订单类型用于新加坡股票。
- 在期权链中增加子市场权限校验。
- 显示股票无法交易的详细原因。

### 轮证
- 添加轮询功能以实时更新数据。

### 成分股
- 为成分股添加轮询功能，确保数据自动更新。

## 优化

### 交易
- 当持仓为负时，显示“买入平仓”。
- 确认订单弹窗中显示保证金估算金额。
- 在持仓为负数时，显示可卖数量为 0。
- 取消选择日期时重置订单有效期限。
- 使用 `window.prompt` 显示交易通知。

### 经纪席位
- 当经纪人数量为 0 时，仍然显示经纪人名称。

### 图表
- 修复 5 日时间线工具提示的“前收盘价”数据错误。
- 修复 Windows 平台上的缩放手势灵敏度问题。
- 优化工具提示的时间日期格式规则。

### 订单
- 确保在订单表格中显示正确的触发状态。

### 分时数据
- 修复分时图历史数据无法更新的问题。

### 期权链
- 修复重置期权链布局时未正常渲染的问题。

### 股票报价
- 优化小屏幕下字段顺序。
- 隐藏无数据时的分配信息。
- 隐藏香港 ETF 的“流通市值”字段。

## 其他

### 命令面板
- 更新命令搜索工具的布局重置命令支持。
- 为命令搜索工具更新占位文本提示。
