# v0.2.2-preview.5  <Badge type="warning" text="preview" />

_发布日期： 2025-05-22_

## 新增

- 为用户设置菜单添加图标。
- 将主题切换移动到设置选项中，优化用户体验。

## 优化

### 交易
- 修复选择有效期后交易时段展示错误的问题。
- 修复有效期下拉框禁用状态时效的问题。
- 优化数字输入框的事件处理，解决输入事件丢失的问题。
- 修复日历控件无法正常显示的问题。

### 图表
- 优化小数据量下的处理逻辑，提高图表展示的精确性。

## 其他
- 改善功能层级逻辑和细节，用于更简洁的界面操作。
