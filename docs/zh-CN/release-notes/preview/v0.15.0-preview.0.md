# v0.15.0-preview.0  <Badge type="warning" text="preview" />

_发布日期： 2026-04-20_

### 新增

- **交易**：新增止盈止损单和平仓功能。
- **交易**：新增美股碎股交易支持。
- **交易**：新增订单导出功能，支持按条件筛选后导出。
- **图表交易**：支持下止盈止损单和快捷平仓。
- **个股**：新增期权计算器面板。
- **登录**：全新的登录和注册界面，支持短信、密码、扫码登录多种方式。
- **搜索**：支持任意界面直接输入字母或数字触发股票搜索。

### 优化

- **反馈**：新增反馈时上传图片的支持。
- **个股**：支持关注股票时快捷加入自定义自选分组。
- **图表**：支持触控板双指捏合手势缩放图表。
- 全局搜索支持期权合约的搜索。
- 优化行情断网后的重连处理。
- 自选股列表支持行情闪烁设置。
- 持仓表格新增「持仓比例」。
