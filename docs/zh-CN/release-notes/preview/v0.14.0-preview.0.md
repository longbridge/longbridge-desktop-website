# v0.14.0-preview.0  <Badge type="warning" text="preview" />

_发布日期： 2026-03-23_

### 新增

- **图表**：新增筹码分布（CYQ）指标，用于分析市场成本结构。
- **图表**：优化图表指标设置和管理功能，提高编辑效率。
- **交易**：订单预览新增卡券抵扣功能，支持使用优惠卡券抵扣交易费用。
- **交易**：支持港股做空。
- **行情**：个股详情新增新股信息展示。
- **资产**：新增「现金明细」。

### 优化

- **设置**：优化主题外观设置交互。
- 补充 Windows 平台缺失的快捷键操作。
- 交易提交按钮新增 Loading 状态，防止重复提交。
- 修复盘前交易时段期权权利金计算错误的问题。
- 修复登录时用户账户列表为空导致的潜在崩溃问题。
