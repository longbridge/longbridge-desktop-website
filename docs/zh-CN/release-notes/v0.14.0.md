# v0.14.0

_发布日期：2026-03-30_

### 新增

- **图表**：新增筹码分布（CYQ）指标，直观展示不同价位的持仓分布情况。

- **图表**：新增图表内指标配置面板，支持直接在图表中设置和管理指标。
<video src="https://assets.lbctrl.com/uploads/8b589035-64a8-4eff-a60e-4be90646c591/indicator.mp4" type="video/mp4" autoplay muted loop>Your browser does not support the video tag.</video>
- **交易**：下单预览界面支持使用交易卡券抵扣费用。
- **行情**：个股详情新增新股信息展示。
- **资产**：新增「现金明细」。
<video src="https://assets.lbctrl.com/uploads/1118d46c-7088-4702-b7d3-9f5f84006871/cash.mp4" type="video/mp4" autoplay muted loop>Your browser does not support the video tag.</video>

### 优化

- **设置**：更方便的主题外观设置交互。
- 补充 Windows 平台缺失的快捷键操作。
- 优化交易下单流程，防止重复提交。
- 修复盘前交易时段期权权利金计算错误的问题。
- 修复登录时用户账户列表为空导致的潜在崩溃问题。
- 修复字体放大后，逐笔成交列表的排版显示错误。
- 优化地区检测，补充断网重连恢复后检测的场景。
