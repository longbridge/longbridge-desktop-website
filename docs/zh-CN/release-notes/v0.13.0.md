# v0.13.0 

_发布日期： 2026-03-09_

### 新增

- 新增设置面板。
- **个股**：新增「估值」分析面板。
- **自选**：新增自选股「多股同列」功能。
- **行情**：期权支持延长时段（盘前）。
- **交易**：当日订单支持「快捷改单」。
- **资产**：全新的「资金记录」。

### 优化

- 调整用户菜单至标题栏，方便查看当前用户信息。
- 修复冬/夏令时切换图表时间轴显示错误的问题。
- 修复 Windows 平台可能出现的启动异常问题。
- 「历史订单」新增「方向」筛选项。
- 优化「历史订单」日期范围默认值。
- 「持仓」表格默认按市值排序。
- 优化交易键盘下单操作。
- 更新交易表单样式细节。
- 优化行情特殊交易状态显示。
- 自选股过滤空分组。
- 优化热力图样式细节。
