# v0.11.0 

_发布日期： 2026-01-08_

### 新增

- **图表：** 新增「画线」功能。
- **自选列表：** 支持自定义分组。
- **浏览器窗口：** 新增内置浏览器。
- **个股：** 新增分析师观点。
<img width="1792" height="1162" alt="SCR-20260112-ocay" src="https://assets.lbctrl.com/uploads/d4f3bf09-8e1c-40b9-817f-def10c72f617/scr-20260112-obug.png" />

- **持仓：** 新增持仓详情界面。

### 优化

#### 交易

- 布局优化，支持单列、多列自适应布局。
- 下单支持切换「数量」和「金额」选项。
- 调整「二次确认」和「金额预览」模态框尺寸。

#### 资产

- 表格自定义设置优化交互和同步机制。
- 「持仓」表格点击买入或卖出时呼出「快捷交易」窗口。
- 优化「融资状态」界面细节。
- 优化未开户/未入金/未输入交易密码等账户状态下的交互和插画细节。
<img width="1792" height="1162" alt="SCR-20260112-obws" src="https://assets.lbctrl.com/uploads/620b272f-26aa-4781-baa6-be40b6a0a88a/scr-20260112-obws.png" />

#### 行情

- 行情等级变化时刷新行情数据。
- 补充在网络恢复时刷新行情数据的机制。
- 持久化「涨跌颜色」设置，避免启动时可能出现的闪烁。
- 优化「买卖盘」、「经纪席位」档位选择功能。

#### 其他

- 使用等宽数字字体。

### 修复

- 修复启动时标签页初始化异常导致的崩溃问题。
- 修复日期筛选器意外的文本换行问题。
- 修复「自动更新」可能出现的更新状态异常。
