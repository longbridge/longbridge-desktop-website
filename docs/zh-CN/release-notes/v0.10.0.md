# v0.10.0

_发布日期： 2025-12-08_

### 新增

- **资产**：在资产标签页新增「当日成交统计」面板，帮助你更直观地掌握今日盈亏数据。
  <img src="https://assets.lbctrl.com/uploads/441996db-d031-4ff0-90c2-1143d6372ca8/portfolio.webp" alt="Today's statistics" width="100%">
- **个股**：个股标签页支持调整后的布局同步到其他个股标签，以便你在不同个股标签页之间保持一致的浏览体验。

### 优化

#### 主题

- 优化了默认主题及其他主题的配色，让界面看起来更加美观。
  <img src="https://assets.lbctrl.com/uploads/faa3c0a6-5a9c-4f53-84d3-71f6ab5fc199/theme.webp" alt="Theme" width="100%">
  > 主题：Default Light / Default Dark
- 优化了主题模式切换逻辑，支持在选中的主题模式下进行亮/暗主题切换。
- 修复了热力图颜色在默认主题中显示不正确的问题。

#### 图表

- 优化了图表设置界面的交互样式，提供更直观的图表配置体验。
- 优化了切换时间范围时的数据时间戳重置逻辑。
- 优化了图表主数据加载后的通知机制。
- 优化了标题显示方式，更加简洁。
- 优化了分时图背景透明度细节。
- 修复了在图表中右键点击时没有响应的问题。
- 修复了加载更多数据时绘图形状丢失的问题。

#### 交易

- 优化了今日订单面板中的改单弹窗交互，现在以独立的窗口显示，并支持自由的拖拽操作。
- 优化了交易窗口高度，并调整了面板布局样式。
- 修复了买卖盘有行情变动时内容闪烁的问题。
- 修复了附加订单撤单和改单的相关问题。
- 修复了替换订单时余额信息预览不正确的问题。
- 修复了订单重复错误通知的问题。

#### 登录

- 优化了登录界面的背景图片，提升用户登录时的视觉体验。
- 新增国家/地区列表中搜索功能，方便快速定位目标国家/地区，并支持按字母分组显示。
- 修复了登录窗口关闭时更新状态错误的问题。

#### 个股

- 优化了股票百科的章节样式和内边距，让内容看起来更加整齐。
- 优化了个股人物信息文字截断逻辑，支持显示更多字符。
- 优化了个股资讯条目显示，修复可能溢出的问题。

#### 自动更新

- 优化了更新成功和失败状态的图标显示。
- 优化了需要重启的对话框类型为确认框，以达到更好的用户体验。

#### 其他

- 新增了在标签页右键菜单中选择「恢复为默认布局」功能，方便你快速恢复标签页的默认布局。
- 优化了持仓/历史订单/今日订单等表格的双击选中交互问题，防止误触发单元格选择。
- 优化了「市场」 - 「版块」标签页面的布局，去掉交易面板仅保留必要的组件。让其更专注于数据的展示，提升使用体验。
