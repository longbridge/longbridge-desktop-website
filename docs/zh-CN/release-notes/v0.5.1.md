# v0.5.1 

_发布日期： 2025-08-07_

### 优化

- 修复设置期权链表头列时可能导致应用异常退出的问题，确保调整显示设置时的稳定性。
- 修复在 Windows 上登录界面点击验证码可能引发的应用异常退出问题，提升登录流程稳定性。
- 修复部分系统因缺失默认字体导致无法启动的问题，确保应用正常运行。
