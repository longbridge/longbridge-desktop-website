# v0.10.0

_發布日期：2025-12-08_

### 新增

- **主题**：引入新的品牌主題色，讓應用程式看起來更加美觀。
  <img src="https://assets.lbctrl.com/uploads/faa3c0a6-5a9c-4f53-84d3-71f6ab5fc199/theme.webp" alt="Theme" width="100%">
  > 主題：Default Light / Default Dark
- **資產**：在資產標籤頁新增「當日成交統計」面板，幫助你更直觀地掌握今日盈虧數據。
  <img src="https://assets.lbctrl.com/uploads/441996db-d031-4ff0-90c2-1143d6372ca8/portfolio.webp" alt="Today's statistics" width="100%">
- **個股**：個股標籤頁支持調整後的佈局同步到其他個股標籤，以便你在不同個股標籤頁之間保持一致的瀏覽體驗。

### 優化

#### 主題

- 優化了主題模式切換邏輯，支持在選中的主題模式下進行亮/暗主題切換。
- 修復了熱力圖顏色在默認主題中顯示不正確的問題。

#### 圖表

- 優化了圖表設置界面的交互樣式，提供更直觀的圖表配置體驗。
- 優化了切換時間範圍時的數據時間戳重置邏輯。
- 優化了圖表主數據加載後的通知機制。
- 優化了分時圖背景透明度細節。
- 修復了在圖表中右鍵點擊時沒有響應的問題。
- 修復了加載更多數據時繪圖形狀丟失的問題。

#### 交易

- 優化了今日訂單面板中的改單彈窗交互，現在以獨立的窗口顯示，並支持自由的拖拽操作。
- 優化了交易窗口高度，並調整了面板佈局樣式。
- 修復了買賣盤有行情變動時內容閃爍的問題。
- 修復了附加訂單撤單和改單的相關問題。
- 修復了替換訂單時餘額信息預覽不正確的問題。
- 修復了訂單重複錯誤通知的問題。

#### 登錄

- 新增國家/地區列表中搜索功能，方便快速定位目標國家/地區，並支持按字母分組顯示。
- 修復了登錄窗口關閉時更新狀態錯誤的問題。

#### 個股

- 優化了股票百科的章節樣式和內邊距，讓內容看起來更加整齊。
- 優化了個股資訊條目顯示，修復可能溢出的問題。

#### 其他

- 新增了在標籤頁右鍵菜單中選擇「恢復為默認佈局」功能，方便你快速恢復標籤頁的默認佈局。
- 優化了持倉/歷史訂單/今日訂單等表格的雙擊選中交互問題，防止誤觸發單元格選擇。
- 優化了「市場」 - 「版塊」標籤頁面的佈局，去掉交易面板僅保留必要的組件。讓其更專注於數據的展示，提升使用體驗。
