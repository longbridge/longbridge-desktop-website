# v0.13.0 

_發布日期： 2026-03-09_

### 新增

- 新增設定面板。
- **個股**：新增估值分析面板。
- **自選**：新增自選股「多股同列」功能。
- **行情**：期權支援延長時段（盤前）。
- **交易**：當日訂單支援「快捷改單」。
- **資產**：全新的「資金記錄」。

### 優化

- 調整用戶選單至標題欄，方便查看當前用戶資訊。
- 修復冬／夏令時切換圖表時間軸顯示錯誤的問題。
- 修復 Windows 平台可能出現的啟動異常問題。
- 「歷史訂單」新增「方向」篩選項。
- 優化「歷史訂單」日期範圍預設值。
- 「持倉」表格預設按市值排序。
- 優化交易鍵盤落單操作。
- 更新交易表單樣式細節。
- 優化報價特殊交易狀態顯示。
- 自選股過濾空分組。
- 優化熱力圖樣式細節。
