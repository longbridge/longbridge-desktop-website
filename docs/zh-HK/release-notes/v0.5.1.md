# v0.5.1 

_發布日期： 2025-08-07_

### 優化

- 修復設定期權鏈表頭列時可能導致應用異常退出的問題，確保調整顯示設定時的穩定性。
- 修復在 Windows 上登入介面點擊驗證碼可能引發的應用異常退出問題，提升登入流程穩定性。
- 修復部分系統因缺失預設字型導致無法啟動的問題，確保應用正常運行。
