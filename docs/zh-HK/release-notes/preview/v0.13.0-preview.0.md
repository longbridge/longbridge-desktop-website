# v0.13.0-preview.0  <Badge type="warning" text="preview" />

_發布日期： 2026-02-26_

### 新增

- 新增全局設定面板。
- **個股**：新增估值分析面板。
- **自選**：新增自選股多圖表功能。
- **行情**：期權支援延長時段（盤前）。
- **交易**：當日訂單支援「快捷改單」。

### 優化

- 調整用戶選單至標題列，方便查看當前用戶資訊。
- 優化交易鍵盤下單操作。
- 更新交易表單樣式細節。
- 優化行情特殊交易狀態顯示。
- 自選股過濾空分組。
- 優化熱力圖樣式細節。
