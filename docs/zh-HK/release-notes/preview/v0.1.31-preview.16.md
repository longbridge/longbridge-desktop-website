# v0.1.31-preview.16 <Badge type="warning" text="preview" />

## 新增

- 支援股票交易的最新交易數據推送時，僅顯示最新的一筆交易，避免多筆交易混亂。
- 提供調整 BMP 權限和優化熱圖遮罩位置的功能。

## 優化

- 優化股票交易平台的整體性能，確保數據更新更加穩定和快速。
- 提高熱圖顯示精度，圖形更加清晰。

## 其他

- 修復了部分情況下可能導致數據顯示異常的 bug。
- 改善界面互動，提升用戶體驗。
