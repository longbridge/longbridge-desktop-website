# v0.13.0-preview.1  <Badge type="warning" text="preview" />

_發布日期： 2026-03-05_

### 新增
- 增加在 Title Bar 中的遮擋方法。
- 多圖表支援使用 `WeakEntity`。
- 更新 `QuoteAppConfig` 的刷新機制。

### 優化
- 優化 Action Bar 中長暱稱的顯示。
- 修改 Counter Search 的下拉寬度及錨點方向。
- 更新訂單顯示範圍為昨天。
- 為過去的訂單表格添加操作篩選器。

### 其他
- 初始化 WebSocket 在異步應用上下文更新後運行。
- 修復貿易問卷確認對話框無法顯示的問題。
- 更新用戶反饋描述內容。
