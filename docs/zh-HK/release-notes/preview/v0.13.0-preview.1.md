# v0.13.0-preview.1  <Badge type="warning" text="preview" />

_發布日期： 2026-03-05_

### 新增
- 資產：全新的「資金明細」。

### 優化

- 修復 Windows 平台可能出現的啟動異常問題。
- 修復 Windows 平台標題列按鈕點擊失效的問題。
- 「歷史訂單」新增「方向」篩選項。
- 優化「歷史訂單」日期範圍的預設值。
- 優化股票選擇下拉框的鍵盤操作體驗。
- 修復「期權鏈」盤前價格顯示問題。
- 「持倉」表格預設按市值排序。
- 修復部分二次確認對話框鍵盤快捷鍵失效的問題。
