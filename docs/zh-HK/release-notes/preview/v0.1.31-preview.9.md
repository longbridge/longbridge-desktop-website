# v0.1.31-preview.9  <Badge type="warning" text="preview" />

## 新增

- 優化股票行業列表排序，顯示更清晰。
- 股票市場狀態欄支持地區篩選。

## 優化

- 改進交易詳情頁面，使信息展示更加精確與清晰。
- 更新市場指數彈出菜單的定位方式，更加準確。
- 修復交易取消後無法恢復有效狀態的問題。
- 提升市場趨勢和熱圖的提示信息展示效果。
- 將技術指標的開關功能移至側邊欄，優化操作流程。
- 從伺服器端獲取股票名稱確保名稱正確無誤。
- 優化圖表上的做空 Y 軸標籤，將精度調整為小數據點後 3 位。
