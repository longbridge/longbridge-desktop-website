# v0.10.0-preview.0  <Badge type="warning" text="preview" />

_發布日期： 2025-11-20_

### 新增

- **資產**: 在資產標籤頁新增「当日成交統計」面板，幫助您更直觀地掌握今日盈虧數據。

### 優化

#### 交易

- 優化了交易窗口顯示方式，從彈窗模式改為獨立窗口模式，提供更靈活的多屏幕操作體驗。
- 優化了下單後的狀態通知機制，確保在交易激活後才進行提醒。
- 修復了訂單重複錯誤通知的問題，避免誤導性的重複提醒。
- 修復了訂單篩選器中清除股票代碼不生效的問題。

#### 账号与登录

- 優化登錄界面國家/地區選擇列表搜索功能，使查找和選擇國家或地區更加便捷。
- 修復了登錄窗口關閉時更新狀態異常的問題。

#### 其他

- 優化了分時圖背景透明度細節，提升視覺效果。
- 優化了自由佈局標籤頁初始化時機，提升自定義工作區的加載速度。
- 應用更新版本時顯示了成功和失敗的狀態圖標，讓更新進度更直觀。
- 修復了網絡重連後市場指數行情詳情未重新加載的問題，確保數據始終保持最新。
