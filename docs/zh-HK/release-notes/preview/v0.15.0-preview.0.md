# v0.15.0-preview.0  <Badge type="warning" text="preview" />

_發布日期： 2026-04-20_

### 新增

- **交易**：新增止盈止蝕單及平倉功能。
- **交易**：新增美股碎股交易支援。
- **交易**：新增訂單匯出功能，支援按條件篩選後匯出。
- **圖表交易**：支援下止盈止蝕單及快捷平倉。
- **個股**：新增期權計算機面板。
- **登入**：全新登入及註冊介面，支援短訊、密碼、掃碼登入等多種方式。
- **搜尋**：支援任何介面直接輸入字母或數字觸發股票搜尋。

### 優化

- **意見回饋**：新增回饋時上載圖片的支援。
- **個股**：支援關注股票時快捷加入自訂自選分組。
- **圖表**：支援觸控板雙指揑合手勢縮放圖表。
- 全域搜尋支援期權合約搜尋。
- 優化行情斷網後的重連處理。
- 自選股列表支援報價閃爍設定。
- 持倉表格新增「持倉比例」。
