# v0.11.0-preview.0  <Badge type="warning" text="preview" />

_發布日期： 2025-12-11_

### 新增

#### 多視窗支援關鍵資產功能

在「資產」標籤頁中出入金、貨幣兌換、資金記錄等在內的核心功能，現支援使用獨立視窗形式開啟，以提升操作效率且不干擾主視窗的使用體驗。

#### 全新的資產頁驗證互動

當你切換到「資產」標籤頁驗證交易密碼或者你未入金、未開戶時會顯示一個更輕量的介面，以優化現在同一個標籤頁多個面板同時顯示重複校驗資訊的不佳體驗。

#### 一鍵查看當前股票訂單與持倉

應用現在支援在當日訂單、歷史訂單、持倉等面板右上角，點擊當前標籤瀏覽的個股按鈕。一鍵篩選出個股的對應的訂單和持倉資訊，以優化現在需要多次點擊才能查到的個股資訊。

#### 其他

- 優化訂單篩選時提供快捷篩選期權的功能，使用戶可以更快速地在股票和期權之間切換。
- 優化了小尺寸螢幕下的行情面板的顯示效果，保持最小的顯示空間，避免內容被壓縮。
- 「資產」-「當日成交統計」新增了對加密貨幣帳戶的支援。
- 修復了在 Linux 系統應用在部分場景異常退出的問題。
- 修復了市價訂單的訂單詳情歷史價格顯示問題。
