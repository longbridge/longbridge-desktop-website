# v0.10.0-preview.1  <Badge type="warning" text="preview" />

_發布日期： 2025-11-27_

### 優化

#### 圖表

- 優化了圖表設置界面的交互樣式，提供更直觀的圖表配置體驗。
- 修復了圖表滑鼠右鍵點擊時沒有響應的問題。

#### 交易

- 優化了今日訂單面板中的改單彈窗以獨立的視窗顯示，並支援自由的拖曳操作。
- 優化了當日成交統計面板交易金額顯示邏輯，讓不同市場不同幣種的金額顯示更加合理。
- 修復了改單時餘額資訊預覽不正確的問題。
- 修復了買賣盤有行情變動時內容閃爍的問題。

#### 主題

- 優化了預設主題及其他主題的配色，讓介面看起來更加美觀。
- 優化了登入介面的背景圖片，提升使用者登入時的視覺體驗。
