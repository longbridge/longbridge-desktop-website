# v0.11.0-preview.2  <Badge type="warning" text="preview" />

_發布日期： 2025-12-31_

### 新增

- 個股「財務」支援查看更多歷史數據。
- 新增「關注清單」分組管理。

### 優化

- 網絡恢復後自動重新整理期權鏈。
- 優化交易表單渲染，避免畫面抖動。
- 修復 WebView 登出後無法正確重新啟動客戶端的問題。
- 調整深度視圖樣式細節。
- 圖表價格推送時更新數據結束時間。
- 瀏覽器模組增加重新整理功能，在移除 WebView 時保證功能一致。
- 盈虧趨勢圖支援圖表動態高度調整，提升渲染效率。
- 修復 WebView 關閉後 Action Bar 失去焦點的問題。
- 更新「快捷交易」快捷鍵為 `cmd + t`。
