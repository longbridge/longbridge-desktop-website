# v0.14.0-preview.0  <Badge type="warning" text="preview" />

_發布日期： 2026-03-23_

### 新增

- **圖表**：新增籌碼分佈（CYQ）指標，用於分析市場成本結構。
- **圖表**：優化圖表指標設定及管理功能，提升編輯效率。
- **交易**：訂單預覽新增卡券抵扣功能，支援使用優惠卡券抵扣交易費用。
- **交易**：支援港股沽空。
- **行情**：個股詳情新增新股資訊展示。
- **資產**：新增「現金明細」。

### 優化

- **設定**：優化主題外觀設定互動體驗。
- 補充 Windows 平台缺失的快捷鍵操作。
- 交易提交按鈕新增 Loading 狀態，防止重複提交。
- 修復盤前交易時段期權權利金計算錯誤的問題。
- 修復登入時用戶賬戶列表為空導致的潛在崩潰問題。
