# v0.4.0-preview.0  <Badge type="warning" text="preview" />

_發布日期： 2025-06-19_

### 優化

- 为交易弹窗增加圆角效果，提升视觉体验。
- 优化自动更新失败时的错误提示，清晰明了，便于快速定位问题。
- 修复侧边栏颜色显示异常，确保主题风格一致。
- 修复子窗口历史操作可能导致的应用崩溃，提升使用稳定性。
