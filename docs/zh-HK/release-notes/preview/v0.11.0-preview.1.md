# v0.11.0-preview.1  <Badge type="warning" text="preview" />

_發布日期： 2025-12-18_

### 優化

- 持倉面板新增買入 / 賣出快捷交易按鈕，現在你可以在持倉面板直接發起交易。
- 優化了表格列寬設定的持久化功能，你對表格的設定會被保存並在下次打開時自動恢復。
- 美股期權新增了在盤前交易時間的交易支援。
- 優化了行情等級變化時的資料準確性。
- 優化了個股圖表中價格更新速度，現在更快更準確。
- 修復了出入金、貨幣兌換等功能需要驗證交易密碼時未及時提示用戶的問題。
- 修復了 Linux 或 Windows 上登出失敗的問題。
