# v0.8.0-preview.1  <Badge type="warning" text="preview" />

_發布日期： 2025-10-09_

#### 優化

- 優化全局歷史列表的樣式及對齊方式，提升視覺體驗。
- 優化操作過濾選項以及操作列的對齊樣式，提高訂單表格的易用性。
- 修復訂單獲取失敗時未顯示錯誤提示的問題，便於快速定位問題。
