# v0.9.0-preview.1  <Badge type="warning" text="preview" />

_發布日期： 2025-11-06_

### 優化

* 優化期權鏈表格的雙擊跳轉功能，便於快速切換至圖表視圖進行技術分析。
* 優化自選股列表在不同視窗大小下的顯示效果，提升瀏覽體驗。
* 優化自訂標籤頁的捲動條顯示邏輯，使介面更簡潔。
* 優化自訂標籤頁中面板的圓角顯示，提升介面一致性。
* 優化表格捲動的響應速度，捲動操作更即時生效。
* 調整附加訂單時間選擇器的顯示位置，居中呈現。
* 修復部分情況下關閉彈窗後介面無法回應的問題，確保操作順暢不中斷。
* 修復登入介面在登入資訊過期後未能提示重新啟動的問題，提升登入體驗的可靠性。
