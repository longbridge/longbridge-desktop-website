# v0.12.0-preview.0  <Badge type="warning" text="preview" />

_發布日期： 2026-01-22_

### 新增

- **資產**: 新增「現金明細」面板。
- **圖表**: 指標加載器優化，支援「參數」和「顏色」設定。
- 新增「訂單」、「持倉」匯出功能。

### 優化

- 修復期權圖表價格變動時可能不更新的問題。
- 優化網絡異常處理，避免意外的登出。
- 改進自選列表分組管理器介面。
- 修正交易預估金額資訊顯示錯誤。
- 優化自動更新交互，顯示安裝進度。
- 優化圖表手勢交互，避免意外的抖動。
- 優化介面文案，統一術語。
- 修復表格設定重置功能失效的問題。
- 在持倉面板中停用當前賬戶訂單篩選。
- 修復部分系統配置拉取異常的問題。
- 修復特殊交易狀態的漲跌顏色顯示錯誤的問題。
