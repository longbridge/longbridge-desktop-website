# v0.9.0 

_發布日期： 2025-11-17_

### 新增

- **交易**：支援於圖表中直接下單與調整訂單，提升操作效率與精準度。
  <video src="https://assets.lbctrl.com/uploads/85a18365-8fcf-47ef-a121-cb841414b7ee/chart-trading.mp4" type="video/mp4" autoplay muted loop>Your browser does not support the video tag.</video>
- **股票簡況**：個股頁面新增股票簡況標籤，便於快速查閱公司資訊與核心指標。
  <video src="https://assets.lbctrl.com/uploads/9f7a71ad-8d31-45be-b3fc-4498b5a411bf/stock_wiki.mp4" type="video/mp4" autoplay muted loop>Your browser does not support the video tag.</video>
- **行情訂閱**：支援於應用內直接訂閱即時行情，免切換平台，簡化操作流程。
  <video src="https://assets.lbctrl.com/uploads/a21b16db-b3d5-4cc8-ab6c-946e11d657b7/quote_subscribe.mp4" type="video/mp4" autoplay muted loop>Your browser does not support the video tag.</video>

### 優化

- 修復置頂新聞未顯示在列表頂部的問題。
- 優化盤後交易類型處理，修復香港股票替換附加訂單失敗的問題。
- 優化訂單推送資料接收順序，確保資訊按時序正確呈現。
- 優化期權策略選單英文顯示，使其更為簡潔明瞭。
- 優化自訂頁面中自選列表欄位寬度自適應表現。
- 優化虛擬資產行情欄位顯示，支援查看專屬行情數據。
- 優化網路重連後市場指數行情詳情自動重新載入機制，確保數據即時更新。
- 修復訂單篩選器中清空股票代碼功能無效的問題。
- 修復自訂搜尋中股票資訊裁切異常的問題。
- 修復期權交易中預估金額計算錯誤的問題。
- 修復期權交易中缺失貨幣資訊的問題。
- 修復複製訂單時交易時段選項遺失的問題。
- 修復預估買入限價金額計算不準確的問題。
- 修復股票停牌時仍顯示入場點的問題。
- 修復複製訂單時交易時段錯誤的問題。
- 修復圖表交易中預估金額計算異常的問題。
- 修復驗證碼輸入過程中出現不必要密碼重設提示的問題。
- 修復新加坡市場預設訂單有效期設置錯誤的問題。
- 修復行情時間戳顯示錯誤的問題。
- 修復附加訂單時間選擇器未置中顯示的問題。
- 修復持倉資訊顯示錯誤的問題。
- 修復雙擊期權表格未能切換至圖表標籤頁的問題。
- 修復小螢幕裝置下自訂工作區導覽滾動異常的問題。
- 修復關閉自訂佈局中其他標籤頁後當前標籤頁失去啟用狀態的問題。
- 修復行情等級變化後深度行情總覽資料缺失的問題。
- 修復替換訂單時訂單詳情視窗中缺少欄位的問題。
- 修復表格設定中勾選框狀態顯示錯誤的問題。
- 修復價格輸入框未自動帶入行情價格的問題。
- 修復輸入框於特定場景下導致應用異常退出的問題。
