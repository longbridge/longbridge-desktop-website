# v0.14.0 

_發布日期： 2026-03-30_

### 新增

- **圖表**：新增籌碼分佈（CYQ）指標，直觀展示不同價位的持倉分佈情況。  
- **圖表**：新增圖表內指標設定面板，支援直接於圖表中設定及管理指標。  
- **交易**：下單預覽介面支援使用交易卡券抵扣費用。  
- **交易**：支援港股沽空。  
- **行情**：個股詳情新增新股資訊展示。  
- **資產**：新增「現金明細」。

### 優化

- **設置**：優化主題外觀設置操作，使用更便捷。  
- 補充 Windows 平台缺失的快捷鍵操作。  
- 優化交易下單流程，防止重複提交。  
- 修復盤前交易時段期權權利金計算錯誤的問題。  
- 修復登入時使用者帳戶列表為空導致的潛在當機問題。  
- 修復字體放大後逐筆成交列表排版顯示錯誤。  
- 優化地區檢測機制，補充斷網重連後的檢測場景。
