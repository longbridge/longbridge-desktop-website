# v0.13.0-preview.0  <Badge type="warning" text="preview" />

_Release Date: 2026-02-26_

### New Features

- Added global settings panel.
- **Stocks**: Added valuation analysis panel.
- **Watchlist**: Added multi‑chart feature for watchlist stocks.
- **Quote**: Options now support extended trading hours (pre‑market).
- **Trade**: Added “Quick Amend Order” for today’s orders.

### Improvements

- Moved the user menu to the title bar for easier access to user information.
- Optimized trading keyboard order placement.
- Updated styling details for the trading form.
- Improved display of special trading statuses in market data.
- Filter out empty groups in the watchlist.
- Refined visual details of the heatmap.
