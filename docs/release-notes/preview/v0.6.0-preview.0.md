# v0.6.0-preview.0  <Badge type="warning" text="preview" />

_Release Date: 2025-08-07_

### New Features

- **Attached Orders**: Added support for attached orders in trading, enabling broader order execution scenarios.

### Improvements

- Improved rejection reason display for orders, offering clearer and more accurate feedback.
- Optimized layout update notifications to appear only once per layout category, reducing interruptions.
- Unified the options chain settings into a single table-style popup for more intuitive configuration.
- Enhanced net flow label display for better information clarity.
- Adjusted price input validation logic to align with minimum price increment, ensuring input accuracy.
- Fixed errors in quote conversion details to maintain data correctness.
- Resolved exchange rate calculation issues in quote conversion to avoid deviations.
- Fixed overlapping table borders to improve visual clarity.
- Fixed unresponsive deposit dialog on Linux systems, enhancing cross-platform compatibility.
- Fixed theme changes not saving to local storage, ensuring user preferences are retained.
