# v0.13.0-preview.1  <Badge type="warning" text="preview" />

_Release Date: 2026-03-05_

### Added
- Introduced the occlude method to the Title Bar.
- Enabled multi-chart support using `WeakEntity`.
- Enhanced refresh mechanism for `QuoteAppConfig`.

### Optimized
- Improved the display of long nicknames in the Action Bar.
- Updated dropdown width and anchor direction in Counter Search.
- Adjusted the order display range to include yesterday.
- Added action filters for the past order table.

### Others
- Initialized WebSocket after the asynchronous app context update.
- Fixed the trade questionnaire confirmation dialog not displaying issue.
- Updated the description for user feedback.
