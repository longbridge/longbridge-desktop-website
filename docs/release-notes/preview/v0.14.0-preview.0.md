# v0.14.0-preview.0  <Badge type="warning" text="preview" />

_Release Date: 2026-03-23_

### New Features

- **Chart**: Added position overview (CYQ) indicator for analyzing market cost structure.
- **Chart**: Optimized chart indicator settings and management for improved editing efficiency.
- **Trading**: Added coupon deduction feature to order preview, supporting fee deduction via promotional coupons.
- **Trading**: Added support for Hong Kong stock short selling.
- **Market Data**: Added IPO information display to individual stock details.
- **Assets**: Added "Cash Details".

### Improvements

- **Settings**: Optimized theme and appearance settings interaction.
- Added missing keyboard shortcuts for Windows platform.
- Added loading state to trade submission button to prevent duplicate submissions.
- Fixed incorrect options premium calculation during pre-market trading sessions.
- Fixed potential crash caused by empty user account list during login.
