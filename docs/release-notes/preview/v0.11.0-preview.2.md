# v0.11.0-preview.2  <Badge type="warning" text="preview" />

_Release Date: 2025-12-31_

### New Features

- Stock “Financials” now supports viewing more historical data.
- Added group management for the “Watchlist”.

### Improvements

- Option chains now auto-refresh after the network connection is restored.
- Optimized trade form rendering to avoid visual jitter.
- Fixed an issue where the client could not restart correctly after logging out in a WebView.
- Fine-tuned style details of the depth view.
- When price pushes are received, the chart now updates the data end time.
- Added a refresh function to the browser module and ensured consistent behavior when removing the WebView.
- The P&L trend chart now supports dynamic height adjustment to improve rendering efficiency.
- Fixed an issue where the Action Bar would lose focus after closing a WebView.
- Updated the “Quick Trade” shortcut to `cmd + t`.
