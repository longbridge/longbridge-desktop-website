# v0.11.0-preview.1  <Badge type="warning" text="preview" />

_Release Date: 2025-12-18_

### Optimizations

- Added buy/sell quick trade buttons in the holdings panel, now you can initiate trades directly from the holdings panel.
- Optimized table column settings persistence, your table settings will be saved and automatically restored when you open it next time.
- Added support for US options trading during pre-market hours.
- Optimized data accuracy when quote level changes.
- Optimized price update speed in stock charts, now faster and more accurate.
- Fixed the issue where users were not prompted in time when verifying trade password for functions such as deposits/withdrawals and currency exchange.
- Fixed logout failure on Linux or Windows.
