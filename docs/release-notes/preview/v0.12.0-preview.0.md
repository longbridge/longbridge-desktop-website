# v0.12.0-preview.0  <Badge type="warning" text="preview" />

_Release Date: 2026-01-22_

### New Features

- **Assets**: Added "Cash Details" panel.
- **Chart**: Optimized indicator loader with support for "Parameters" and "Color" settings.
- Added export functionality for "Orders" and "Positions".

### Improvements

- Fixed option chart price update issues.
- Optimized network exception handling to avoid unexpected logouts.
- Improved watchlist group manager interface.
- Fixed incorrect display of trade estimation amount information.
- Optimized auto-update interaction with installation progress display.
- Optimized chart gesture interaction to avoid unexpected jittering.
- Optimized interface copy and unified terminology.
- Fixed table settings reset functionality failure.
- Disabled current account order filter in positions panel.
- Fixed system configuration fetch exception issues.
- Fixed incorrect trend color display for special trade status.
