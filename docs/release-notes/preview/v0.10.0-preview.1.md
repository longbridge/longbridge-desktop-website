# v0.10.0-preview.1  <Badge type="warning" text="preview" />

_Release Date: 2025-11-27_

### Optimizations

#### Chart

- Optimized the chart settings window, providing a more intuitive chart configuration experience.
- Fixed the issue where right-click on the chart had no response.

#### Trade

- Optimized the order modification modal in the Today's Orders panel to display as a window, supporting free drag operations.
- Optimized the transaction amount display logic in the Today's Trades statistics panel, making the amount display more reasonable for different markets and region.
- Fixed the issue where balance information preview was incorrect when modifying orders.
- Fixed the issue where content flickered when there were quote changes in the bid-ask panel.

#### Theme

- Optimized the color scheme of the default theme and other themes, making the interface look more beautiful.
- Optimized the background image of the login window, enhancing the visual experience during user login.
