# v0.9.0-preview.1  <Badge type="warning" text="preview" />

_Release Date: 2025-11-06_

### Improvements

* Enhanced the double-click behavior in the options chain table to seamlessly jump to the chart view, streamlining technical analysis access.
* Improved responsive layout of the watchlist under different window sizes for better viewing experience.
* Refined scrollbar display logic in custom tab pages for a cleaner interface.
* Polished panel corner rendering in custom tab pages for a more consistent appearance.
* Improved table scroll responsiveness, making scroll actions instantly effective.
* Adjusted the position of the time selector in additional order settings to display centered.
* Fixed an issue where the interface became unresponsive after closing certain pop-ups, ensuring uninterrupted interaction.
* Resolved a problem where expired login sessions did not prompt application restart, improving login reliability.
