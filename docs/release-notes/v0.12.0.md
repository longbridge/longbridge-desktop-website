# v0.12.0 

_Release Date: 2026-02-05_

### New Features

- New brand logo.
<img width="402" height="396" alt="new-logo" src="https://assets.lbctrl.com/uploads/80ab481b-d840-488a-beb8-d424bdc85d75/new-logo.png" /><img width="402" height="396" alt="new-logo-dark" src="https://assets.lbctrl.com/uploads/423cb07b-8b1e-424f-b3d8-3210147ce108/new-logo-dark.png" />

- **Indicators**: Support for custom technical indicator configuration and editing.
<img width="1704" height="1144" alt="panel" src="https://assets.lbctrl.com/uploads/0138b4a1-5a59-4eea-9e15-8d03b087c557/indicators.png" />

- **Chart**: Optimized indicator loader with support for "Parameter" and "Style" settings.
<img width="892" height="712" alt="setting" src="https://assets.lbctrl.com/uploads/37bc3d32-b43f-4bfd-84a2-cabf31be1e84/setting.png" />

- **Assets**: Added "Cash Balance" panel.
- Added export functionality for "Orders" and "Positions".

### Improvements

#### Trading

- Enhanced trading keyboard operation experience with support for keyboard shortcuts.
- Improved trade notification experience - notifications now only push to the currently active window to avoid duplicates.
- Added auto-refresh functionality for "Today's Orders" to prevent possible delays in order updates caused by push anomalies.
- Added default status filter for "Historical Orders", defaulting to "Filled" status.
- Fixed incorrect display of estimated trade amount information.

#### Quotes

- Fixed issue where option chart prices might not update when prices change.
- Removed current account order filter option from portfolio panel.

#### Portfolio

- Adjusted minimum width of assets sidebar.

#### Other

- Optimized network exception handling to prevent unexpected logouts.
- Fixed issue where table settings reset functionality was not working.
- Unified neutral color display in analysis module.
- Heatmap optimization with adjustments to some style details and hover border styles.
- Auto-update optimization now uses progress circle to display download progress.
- Fixed issues with retrieving some system configurations.
- Fixed incorrect color display for special trading status price changes.
