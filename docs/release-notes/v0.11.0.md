# v0.11.0 

_Release Date: 2026-01-08_

### New Features

- **Chart:** Added drawing tools functionality.
- **Watchlist:** Added support for custom groups.
- **Browser Window:** Added built-in browser.
- **Stock Detail:** Added analyst opinions.
<img width="1792" height="1162" alt="SCR-20260112-ocay" src="https://assets.lbctrl.com/uploads/d4f3bf09-8e1c-40b9-817f-def10c72f617/scr-20260112-obug.png" />

- **Holdings:** Added holdings detail interface.

### Improvements

#### Trading

- Layout optimization with support for single-column and multi-column adaptive layouts.
- Added support for switching between "Quantity" and "Amount" options when placing orders.
- Adjusted the size of "Confirmation" and "Amount Preview" modal dialogs.

#### Assets

- Optimized interaction and synchronization mechanism for table custom settings.
- Clicking buy or sell in the "Holdings" table now opens the "Quick Trade" window.
- Improved "Margin Status" interface details.
- Optimized interaction and illustration details for account states such as not opened/not funded/trade password not set.
<img width="1792" height="1162" alt="SCR-20260112-obws" src="https://assets.lbctrl.com/uploads/620b272f-26aa-4781-baa6-be40b6a0a88a/scr-20260112-obws.png" />

#### Quotes

- Refresh quote data when quote level changes.
- Added mechanism to refresh quote data when network reconnects.
- Persist "Price Change Color" settings to avoid potential flickering during startup.
- Optimized tier selection for "Bid/Ask" and "Broker Queue".

#### Others

- Use monospaced digit font.

### Bug Fixes

- Fixed crash issue caused by abnormal tab initialization during startup.
- Fixed unexpected text line break in date filter.
- Fixed potential update status anomaly in "Auto Update".
