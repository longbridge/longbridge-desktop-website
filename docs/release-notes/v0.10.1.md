# v0.10.1 

_Release Date: 2025-12-10_

### Optimizations

- Fixed an issue where historical data might fail to load correctly after switching adjustment settings in K-line charts.
- Fixed an issue where filter options were not cleared after clicking "Clear" in the buy/sell direction filter for today's orders and historical orders.
