# v0.10.0

_Release Date: 2025-12-08_

### New Features

- **Portfolio**: Added "Today's Statistics" panel in the Portfolio tab, helping you better understand your daily profit and loss data.
  <img src="https://assets.lbctrl.com/uploads/1464bedd-2762-414a-a89a-6409ad22c338/portfolio.webp" alt="Today's statistics" width="100%">
- **Stock**: Stock tab now syncs layouts to other stock tabs, allowing you to maintain a consistent browsing experience across different stock tabs.

### Optimizations

#### Theme

- Improved color schemes for default and other themes, making the app more visually appealing.
  <img src="https://assets.lbctrl.com/uploads/faa3c0a6-5a9c-4f53-84d3-71f6ab5fc199/theme.webp" alt="Theme" width="100%">
  > Theme: Default Light / Default Dark
- Improved theme mode change logic, supporting light/dark theme toggle within the selected theme mode.
- Fixed incorrect heatmap colors in the default theme.

#### Chart

- Improved chart settings interface with better interaction styles for a more intuitive configuration experience.
- Improved data timestamp reset logic when switching time ranges.
- Improved notification mechanism after chart main data is loaded.
- Improved title display for a more concise look.
- Improved timeshare chart background opacity details.
- Fixed issue where right-clicking in charts had no response.
- Fixed issue where drawing shapes were lost when loading more data.

#### Trading

- Improved order modification dialog interaction in Today's Orders panel, now displayed as an independent window with free drag-and-drop support.
- Improved trading window height and adjusted panel layout styles.
- Fixed content flickering issue when bid/ask quotes changed.
- Fixed issues related to attached order cancellation and modification.
- Fixed incorrect balance information preview when replacing orders.
- Fixed duplicate error notifications for orders.

#### Login

- Improved login interface background image, enhancing visual experience during login.
- Added search functionality in country/region list for quick location, with alphabetical grouping support.
- Fixed update status error when login window is closed.

#### Stock

- Improved stock wiki section styles and padding for neater content presentation.
- Improved text truncation logic for stock profile information, supporting more character display.
- Improved stock news item display, fixing potential overflow issues.

#### Auto Update

- Improved icon display for update success and failure status.
- Improved restart dialog type to confirmation box for better user experience.

#### Others

- Added "Reset layout" option in tab context menu for quick restoration of default tab layout.
- Improved double-click selection interaction in tables such as Holdings/Past Orders/Today's Orders to prevent accidental cell selection.
- Improved layout of "Market" - "Sector" tab page, removing trading panel and keeping only essential components for better focus on data display and improved user experience.
