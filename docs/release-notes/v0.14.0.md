# v0.14.0

_Release Date: 2026-03-30_

### New Features

- **Charts**: Added the Position overview (CYQ) indicator to visualize position distribution across different price levels.  
<video src="https://assets.lbctrl.com/uploads/54103c18-8424-46f5-8668-63d64efa680d/cyq.mp4" type="video/mp4" autoplay muted loop>Your browser does not support the video tag.</video>
- **Charts**: Added an in-chart indicator settings panel, allowing users to configure and manage indicators directly within the chart.  
<video src="https://assets.lbctrl.com/uploads/8b589035-64a8-4eff-a60e-4be90646c591/indicator.mp4" type="video/mp4" autoplay muted loop>Your browser does not support the video tag.</video>
- **Trading**: Order preview now supports applying trading coupons to offset fees.  
- **Market**: Added IPO information display in stock detail pages.  
- **Assets**: Added “Cash Details”.  
<video src="https://assets.lbctrl.com/uploads/1118d46c-7088-4702-b7d3-9f5f84006871/cash.mp4" type="video/mp4" autoplay muted loop>Your browser does not support the video tag.</video>

### Improvements

- **Settings**: Improved theme appearance settings with a more intuitive interaction design.  
- Added missing shortcut key operations on Windows.  
- Optimized the order placement process to prevent duplicate submissions.  
- Fixed incorrect option premium calculation during pre-market trading sessions.  
- Fixed a potential crash issue when the account list was empty during login.  
- Fixed layout issues in the time & sales list when fonts were enlarged.  
- Improved region detection, including scenarios after reconnecting from network disconnection.
