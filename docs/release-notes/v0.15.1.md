# v0.15.1 

_Release Date: 2026-04-24_

### New Features

- Watchlist now supports switching the position of stock name and code.

### Optimizations

- Fixed an issue where the order session for option stop-profit/stop-loss orders was incorrect.
- Fixed an issue where the estimated quantity on the order screen was displayed inaccurately.
