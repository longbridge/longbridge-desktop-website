# v0.15.0 

_Release Date: 2026-04-21_

### New Features

- **Trading**: Added take-profit/stop-loss orders and position closing.
<img width="847" height="361" alt="tpsl" src="https://github.com/user-attachments/assets/489ead27-d5f6-425f-92d5-cdbc09e4996f" />

- **Trading**: Added support for US stock fractional share trading.
<img width="845" height="547" alt="shares" src="https://github.com/user-attachments/assets/843532fe-b7b9-476d-bdbf-23fe5e9fbb84" />

- **Trading**: Added order export functionality with filtering options.
<img width="847" height="362" alt="export" src="https://github.com/user-attachments/assets/efd1077d-443e-446f-b7eb-afef10266b29" />

- **Chart Trading**: Added support for placing take-profit/stop-loss orders and quick position closing.
<img width="848" height="593" alt="chart_tpsl" src="https://github.com/user-attachments/assets/a1596e0d-4707-4215-8870-4b5e2706c5d3" />

- **Stock Details**: Added options calculator panel.
<img width="430" height="606" alt="option" src="https://github.com/user-attachments/assets/07544ae1-c3e6-4150-9290-121f770618b4" />

- **Login**: Redesigned login and registration interface, supporting SMS, password, and QR code login.
- **Search**: Stock search can now be triggered by typing letters or numbers from any screen.

### Improvements

- **Feedback**: Added image upload support when submitting feedback.
<img width="447" height="621" alt="feedback" src="https://github.com/user-attachments/assets/3f5e9d41-3d44-405a-8c08-6eecca311630" />

- **Stock Details**: Added quick option to assign stocks to custom watchlist groups when following.
- **Charts**: Added two-finger pinch gesture support for zooming charts on trackpads.
- Global search now supports options contract search.
- Improved reconnection handling after market data disconnection.
- Watchlist now supports quote flashing settings.
- Added "Position Ratio" to the holdings table.
- Support full-width numbers from input methods.
